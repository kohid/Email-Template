# Images Directory

## Required Images

### placeholder.png
- Dimensions: 800x600px
- Use: Default placeholder image for image widgets
- You can replace this with your own placeholder image

### logo.png
- Dimensions: 60x60px (or any width, height will auto-adjust)
- Use: Plugin logo in the builder header
- Default: Uses your company logo or create a simple logo

## Creating Placeholder Images

You can create these images using:
1. Any image editing software (Photoshop, GIMP, etc.)
2. Online tools like Canva
3. Or use data URIs in the code (inline base64 images)

## Note
The plugin will work without these images, but some UI elements may not display perfectly.
